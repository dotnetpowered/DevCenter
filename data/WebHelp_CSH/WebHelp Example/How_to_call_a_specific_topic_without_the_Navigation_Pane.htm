How to call a specific topic without the Navigation Pane

To open a topic without the navigation pane is exactly like opening the a topic with the navigation pane.  The only difference is that you specify a different window name in your API call.  In the source of the help files you will notice that the properties of the SecondWindow are set to One Pane View.
