How to call a specific topic with the Navigation Pane

To open a specific topic with the Navigation pane displayed, you must specify a window name following the path to your WebHelp start page.  The following would be the syntax if we were to write it out completely without using variables:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm>First", HH_HELP_CONTEXT, 7)'>Click Here to Test</a>

In the Help's source files you will notice that their is a window defined as First.  This is the window that we are telling the help topic to open in.  In this example we have setup the windows properties to use a 'Two Pane' view, this allows us to open the mapped topic while showing the navigation pane.
