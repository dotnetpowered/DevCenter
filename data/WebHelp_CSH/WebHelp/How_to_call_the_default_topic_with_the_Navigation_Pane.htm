How to call the default topic with the Navigation Pane

To open the default topic with the Navigation pane displayed, you must specify only the default start page for your FlasHelp project.  Here is the function used to call the help:

function RH_ShowHelp(hParent, a_pszHelpFile, uCommand, dwData)

Parameter

Data Type

Description

hParent

variable

Reserved for future versions of API. Use "0."

a_pszHelpFile

variable

Help source

For WebHelp/FlashHelp: "<Path to project start page>" (Can be local or on a server)

For WebHelp Pro: "http://<ServerName>/roboapi.asp"

*Optional: To specify a window, use ">WindowName" at the end of this parameter

UCommand

variable

Command

You can use the following constants, which are defined in RoboHelp_CSH.js:

HH_DISPLAY_INDEX Displays Index pane and default topic.

HH_DISPLAY_SEARCH Displays Search pane and default topic.

HH_DISPLAY_TOC Displays Contents pane and default topic.

HH_HELP_CONTEXT Opens topic associated with map ID sent in dwData parameter.

*Note: WebHelp Pro/FlashHelp will always open to the default pane, no matter what is specified in the UCommand variable

dwData

variable

Map ID of the topic displayed (defined in the Edit Map IDs dialog in RoboHelp). To obtain the map ID, the author can export the map file for your programming language (you must use HH_HELP_CONTEXT in the uCommand parameter).

 

The following would be the syntax if we were to write it out completely without using variables:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm", HH_HELP_CONTEXT, 0)'>Click Here to Test</a>
