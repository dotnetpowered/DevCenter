Frequently Asked Questions

Q:  Is it possible to specify in the call what tab (Contents, Index, Search) the help is opened to?

A:  It is not possible to specify this in either WebHelp Pro call or FlashHelp call, the help will always open to the default tab which can be specified inside of RoboHelp.  With WebHelp output you can specify what tap the output is opened to.

Q: Why are my help calls not working?

A:  Make sure you have linked the pages that call the help to the RoboHelp_csh.js file as mentioned in the Setup.
