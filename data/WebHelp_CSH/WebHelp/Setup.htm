Setup

To create context-sensitive Help for a Web page:

  1. Talk to the Help author to determine:

  • Who is responsible for providing the map numbers.

  • If the project already has context-sensitive Help set up.

  • Where the Help system will reside (locally or on a server).

 

  1. Open your Web page(s) in your development tool.
     

  2. Link the page to RoboHelp_csh.js (click to save the file):

    <script  type="text/javascript" language=JavaScript1.2 src="RoboHelp_CSH.js">

    </script>
