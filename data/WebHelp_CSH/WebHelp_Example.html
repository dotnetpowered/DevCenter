Calling Context Sensitive WebHelp From a HTML Page

The examples below will who you how to call FlashHelp Context Sensitively from a HTML Page.  Before getting started I would read the Help file that I have created for this project which you can find Here.

How to call the default topic with the Navigation Pane

This first example shows you how to call FlashHelp opening the project to the default topic.

Here is the syntax used:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm", HH_HELP_CONTEXT, 0)'>Click Here to Test</a>

Click Here to Test

You can also specify to open the default topic and a different pane, such as the index or search pane.

Here is the syntax used to display the Search tab:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm", HH_DISPLAY_SEARCH, 0)'>Click Here to Test</a>

Click Here to Test

Or the syntax used to display the Index tab:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm", HH_DISPLAY_INDEX, 0)'>Click Here to Test</a>

Click Here to Test

How to call a specific topic with the Navigation Pane

Here is the syntax used:

<A HREF='javascript:RH_ShowHelp(0, "WebHelp/WebHelp_Example.htm>First", HH_HELP_CONTEXT, 7)'>Click Here to Test</a>
